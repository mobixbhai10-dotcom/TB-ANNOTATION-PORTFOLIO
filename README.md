# TB Chest X-Ray Annotation Portfolio

## About
Manual annotation of TB Chest X-rays using CVAT.
Annotator: 8 years clinical experience in CT/MRI (MSc MRIT)
## Dataset
Kaggle TB Chest Radiography Database (Tawsifur Rahman)

## Annotation Schema
| Label | Clinical Significance |
|---|---|
| consolidation | Active TB - lobar involvement |
| cavity | Advanced TB - necrotic lung tissue |
| pleural_effusion | TB complication - pleural seeding |
| normal | No active TB findings |

## Tools
- CVAT (app.cvat.ai)
- Format: COCO 1.0
- Method: Manual polygon annotation
- Progress
- Batch 1: 20 images ✅

Annotations informed by 8 years of clinical radiological experience — not just boxes, but radiological reasoning
